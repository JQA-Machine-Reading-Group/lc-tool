LC Capture Tool Help

Close Button

The Close Button will close the current document and return you to the main file listing. Please be aware that the application does not automatically save your session.

Undo/Redo

The Undo button will remove features in the order that they were added. The redo button will roll back undoes, but after you make a new change to the document, the redo history is cleared. Note that all 3 capture modes have separate undo/redo history.

Capture Modes

All of these modes will reject your capture if it looks erroneous (i.e., smaller than 10px)

Letter Capture Mode

This mode lets you capture one letter at a time. Click and drag to draw a bounding box around the character you are trying to capture. After drawing the bounding box, press a keyboard key to label the letter. You cannot capture new letters before labeling the current letter.

You cannot switch to a different mode if you have an unlabeled letter. This will also prevent you from saving.

Word Capture Mode

This is an experimental mode to capture words. Draw a line through all the letter bounding boxes to make a word. You do not need to label these.

Line Capture Mode

This is an experimental mode to capture lines. Draw a line through all the letter bounding boxes in a line to make a line. You do not need to label these.

Mark As Completed

Use this to mark a capture job as completed.

Save

This button sends your capture data to the backend, where it will be cropped and the capture data saved. You should make sure the crops look correctly at first, to make sure things are working as expected on your machine/browser combination. Check the logs for errors.

You can't have unlabeled letters when saving. Additionally, your work is not saved automatically. If you leave the page without saving, you will lose your work.

If you accidentally shutdown your backend, you can just reboot it and it should be able to process the capture data again. You can also use the Download Job Info button to manually save a backup copy of the JSON.

Download Job Info

Use this to manually download the JSON capture data. If something is going wrong with the backend and you don't want to lose your work, save this as diagnostic information. Likely if you send me the JSON and image file (jjb@uconn.edu), I can figure out and fix what is causing the problem.
