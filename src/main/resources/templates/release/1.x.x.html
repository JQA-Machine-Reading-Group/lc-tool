LC Capture Tool 1.0.0-2.0.0 Cumulative Release Notes

Release 1.1.0

  • Desktop-user friendly document panning enabled on capture page. Hold down Ctrl or Shift and drag to pan the page
  • Some resources are now hidden in the navbar until authenticated
  • Document images are now loaded as files instead of as Base64 encoded data
  • Migration Service is now deprecated and will be removed in the next release
  • ImageJob format will be replaced in the next release to accomodate editing another user's document. This is planned to be transparent to the user.
  • Various code cleanup, bugfixes and other enhancements

Release 1.0.1

  • When redirected to a log in page when attempting to access a protected resource, try to redirect back to the correct page after login.
  • Users without an @uconn.edu email can be invited to use the application
